# snaplt - macOS 창 관리 애플리케이션

snaplt은 macOS 창 관리 애플리케이션으로, 마우스를 사용하지 않고도 키보드 단축키나 메뉴바를 통해 쉽고 빠르게 창을 배치할 수 있습니다.

## 주요 기능

- 다양한 레이아웃 (전체화면, 반분할, 1/4분할, 1/3분할, 2/3분할 등)
- 키보드 단축키
- 멀티 모니터 지원
- 직관적인 메뉴바
- 사용자 설정 가능
- 레이아웃 반복 적용

## 설치 방법

웹사이트에서 snaplt.zip 파일을 다운로드하여 설치하세요.

## 주의 사항

바이브 코딩을 연습하며 만든 앱입니다.
