👋소개

snaplt은 macOS 창 관리 애플리케이션으로, 마우스를 사용하지 않고도 키보드 단축키나 메뉴바를 통해 쉽고 빠르게 창을 배치할 수 있습니다. 다양한 화면 분할 레이아웃을 지원하며 여러 모니터 환경에서도 원활하게 작동합니다.

💿설치

  1. 다운로드한 snaplt 앱 파일을 응용 프로그램 폴더로 이동합니다.
  2. 앱을 처음 실행할 때 macOS에서 접근성 권한을 요청합니다. 이 권한은 창을 제어하기 위해 반드시 필요합니다.

🔐접근성 권한 설정

  1. snaplt 앱을 처음 실행하면 접근성 권한 요청 알림이 표시됩니다.
  2. "설정 열기" 버튼을 클릭하여 시스템 환경설정의 개인 정보 보호 및 보안 > 접근성으로 이동합니다.
  3. 목록에서 snaplt 앱을 찾아 체크박스를 선택하여 권한을 부여합니다.
  4. 권한을 부여한 후에 앱을 다시 시작해야 할 수 있습니다.

🎯기본 사용법

메뉴바 사용

  1. 앱을 실행하면 메뉴바에 snaplt 아이콘이 표시됩니다.
  2. 아이콘을 클릭하면 사용 가능한 창 레이아웃 목록이 표시됩니다.
  3. 원하는 레이아웃을 클릭하면 현재 활성화된 창에 해당 레이아웃이 적용됩니다.
  4. 각 레이아웃 옆에는 해당 기능의 단축키가 표시됩니다.

🪟지원하는 레이아웃

📐전체화면

창을 모니터 전체 크기로 확장합니다.

📐좌측/우측 반분할

창을 화면의 왼쪽/오른쪽 절반으로 배치합니다.

📐상단/하단 반분할

창을 화면의 위쪽/아래쪽 절반으로 배치합니다.

📐사분면 배치

좌상단/우상단/좌하단/우하단 1/4 크기로 창을 배치합니다.

📐좌측/우측 2/3

창을 화면 좌측/우측의 2/3 크기로 배치합니다.

📐1/3 분할

좌측/중앙/우측으로 화면의 1/3 크기로 창을 배치합니다.

⌨️키보드 단축키

기본 단축키 설정은 다음과 같습니다:

전체화면: ⌃⌥ return

좌측 반분할: ⌃⌥ ←

우측 반분할: ⌃⌥ →

상단 반분할: ⌃⌥ ↑

하단 반분할: ⌃⌥ ↓

좌상단 1/4: ⌃⌥ I

우상단 1/4: ⌃⌥ O

좌하단 1/4: ⌃⌥ K

우하단 1/4: ⌃⌥ L

좌측 1/3: ⌃⌥ Q

중앙 1/3: ⌃⌥ W

우측 1/3: ⌃⌥ E

좌측 2/3: ⌃⌥ ← ←

우측 2/3: ⌃⌥ → →

⚙️환경설정

환경설정 열기

메뉴바에서 snaplt 아이콘을 클릭한 후 하단의 "환경설정" 버튼을 클릭하거나, 앱이 실행 중일 때 macOS 기본 단축키인 ⌘,(Command + 쉼표)를 사용합니다.

일반 설정

  • 로그인 시 자동 실행: 활성화하면 컴퓨터 시작 시 snaplt이 자동으로 실행됩니다.
  • 화면 정보: 연결된 모든 모니터의 해상도 정보를 확인할 수 있습니다.

단축키 설정

  • 전역 단축키 활성화: 이 옵션을 켜면 전역 단축키가 작동합니다. 끄면 단축키가 비활성화됩니다.
  • 레이아웃별 단축키 설정: 각 레이아웃에 대해 원하는 단축키를 직접 설정할 수 있습니다.
    1. 변경하려는 레이아웃의 단축키 설정 버튼을 클릭합니다.
    2. "입력 중..." 상태가 되면 원하는 키 조합을 입력합니다.
    3. 설정된 단축키를 삭제하려면 오른쪽의 X 버튼을 클릭합니다.

💡팁과 트릭

🖥다중 모니터

snaplt은 자동으로 창이 위치한 모니터를 감지하여 해당 모니터에 맞게 레이아웃을 적용합니다.

🔄레이아웃 반복 적용

메뉴바에서 레이아웃을 선택했을 때 "레이아웃 적용, 계속 반복하시겠습니까?" 알림이 표시됩니다. "예"를 선택하면 같은 레이아웃을 다른 창에 연속해서 적용할 수 있습니다.

⌨️자주 사용하는 레이아웃

가장 자주 사용하는 레이아웃에는 쉽게 기억할 수 있는 단축키를 설정하는 것이 좋습니다.

🔧문제 해결

창이 이동되지 않는 경우:

  • 접근성 권한이 올바르게 설정되었는지 확인합니다.
  • 앱을 종료하고 다시 시작합니다.
  • 시스템 환경설정 > 개인 정보 보호 및 보안 > 접근성에서 snaplt 앱 권한을 확인합니다.

단축키가 작동하지 않는 경우:

  • 환경설정에서 전역 단축키가 활성화되어 있는지 확인합니다.
  • 다른 앱에서 같은 단축키를 사용하고 있지 않은지 확인합니다.
  • 단축키 설정을 새로 지정해봅니다.

특정 앱에서 작동하지 않는 경우:

  • 일부 앱은 보안 또는 내부 구현 이유로 창 제어가 제한될 수 있습니다.
  • 앱이 전체 화면 모드인 경우 일반 모드로 전환 후 시도합니다.
